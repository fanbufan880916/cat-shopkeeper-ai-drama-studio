# GPT-Image-2 漫剧提示词指南

来源：OpenAI 官方 Image generation guide 与 Image generation prompting guide，最后核实日期 2026-07-14。
本地角色设定图规范最后更新：2026-07-16。

- 把提示词写成清楚的制作说明，而不是关键词堆砌。
- 明确主体、环境、构图、相机视角、光线、色彩、材质和用途。
- 编辑图片时先说明必须保留什么，再说明只改变什么。
- 多参考图需要说明每张图的作用，避免模型自行猜测身份和风格来源。
- 对文字内容写出准确文字、位置、字体气质和层级；重要文字仍应预留后期复核。
- 对角色一致性使用稳定、可重复的可见特征，不依赖“和上一张一样”这类模糊说法。
- 复杂画面先锁定身份和构图，再逐轮增加动作、光线或细节。

## 角色设定图（资产）

角色图不是剧情肖像，必须按“角色设计参考拼版”写：

1. 用途先写：角色设定图 / character design reference sheet。
2. 画幅：`3:2` 横向。
3. 版式必须写清：
   - 左侧约 40%：脸部特写，锁定五官、脸型、发型、神态。
   - 右侧约 60%：正面全身、侧面全身、背面全身三视图。
4. 光线与背景：浅灰中性背景 + 柔和均匀棚光。
5. 连续性：明确“左右同一角色，脸部特写与三视图面部完全一致，不要改变脸型/五官/发型/体型/服装”。
6. 禁止：9:16 分镜静帧、港风霓虹、雨夜、胶片诗意、复杂剧情场景。

示例骨架：

```text
角色设定图，角色设计参考拼版，3:2 横向构图。
左侧约40%是同一角色的脸部特写，锁定五官、脸型、发型和神态。
右侧约60%是同一角色的全身三视图并排：正面全身、侧面全身、背面全身。
主体：{年龄/性别/物种}，{脸型五官发型}，{服装主色与材质}，{配饰}。
浅灰中性背景，柔和均匀棚光，干净可比较构图。
左右为同一角色，脸部特写与三视图面部完全一致；不要改变脸型、五官比例、发型、体型、服装配色和关键配饰。
```

## 分镜首帧

分镜首帧才写故事时刻、景别、戏剧光、人物关系和动作起始状态；并引用已锁定角色身份锚点，不要重新发明脸。
