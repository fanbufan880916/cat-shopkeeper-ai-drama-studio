# GPT-Image-2 漫剧生图

## 工作顺序

1. 读取项目、画幅、图片用途和已锁定资产。
2. 为每张参考图指定唯一职责：身份、服装、场景、构图、姿势或材质。说明禁止迁移什么。
3. 按以下顺序写自然语言提示词：
   - 画面用途与叙事时刻
   - 主体固定身份锚点（只写可见特征，不写资产 ID）
   - 动作、表情和人物关系
   - 环境与关键道具
   - 景别、构图、视角和留白
   - 有实际来源的光线、色彩和材质风格
   - 必须延续的连续性
   - 禁止变化项
4. 运行 `python scripts/prompt_lint.py --prompt-file <file> --aspect <ratio>`。
5. 根据 [APIMart 接口](references/apimart-api.md) 生成请求参数。禁止把未支持的 OpenAI 字段传给 APIMart。
6. 用户退回时保留已正确部分，只修改一个主要变量。局部问题优先使用官方通道 mask 重绘。

## 资产纪律

- 同一角色必须在工作台结构化数据中引用相同资产 ID，并使用相同身份锚点，不能每个镜头重新描述长相。
- 资产 ID、项目名、版本号和"用于某项目"等管理信息只放在工作台的资产引用字段，不得写进发送给生图模型的自然语言提示词。
- 资产设定图（角色/场景/道具）使用中性光线和易比较构图；戏剧光线留到镜头首帧。
- 场景用广角空间交代+无人物；道具用干净产品摄影风格。
- 双人或多人画面逐一说明位置、动作、视线和身份，避免特征串位。
- 分镜首帧描述画面开始时的静态可见状态，不提前写完整视频运动过程。
- 否定要求写成明确的"保持/不要改变"，避免堆叠空泛负面词。
- 资产设定图默认使用 GPT-Image-2 官方模型 `gpt-image-2-official`，分辨率 1K，质量 high，单张生成。戏剧光线（霓虹、框景、anamorphic bokeh、颗粒、雨夜反光等）只用于分镜首帧，不用于资产设定图。
- 资产设定图与分镜首帧的提示词风格必须严格区分：资产图只描述"是什么"，分镜首帧才描述"怎么拍"。

## 角色设定图硬规范（必须遵守）

角色资产图默认必须是"角色设计参考拼版 / character design reference sheet"，不是单人特写，也不是剧情分镜。

### 画幅与默认参数

- 画幅固定 `3:2` 横向（左脸 + 右三视图需要横向空间）
- 模型 `gpt-image-2-official`，分辨率 `1k`，质量 `high`，`n: 1`
- 背景：干净浅灰/中性工作室色，无复杂场景、无叙事道具堆
- 光线：柔和均匀棚光；禁止霓虹、雨夜、胶片刻意风格、戏剧框景

### 强制版式（不写进分区字样也不许缺构图）

提示词中必须明确写出以下版式，模型才能稳定执行：

1. **左侧约 40%：脸部特写**
   - 高清晰近景/头肩特写
   - 锁定五官、脸型、发型、肤色、神态与妆容
   - 目的：后续分镜复用时脸不会漂

2. **右侧约 60%：全身三视图**
   - 同一服饰、同一体型、同一发型
   - 必须同时包含：**正面全身、侧面全身、背面全身**
   - 三视图并列、站姿自然、手脚完整、鞋子可见
   - 服装剪裁、配色、配饰要看得见

### 身份锚点要求

- 身份锚点只写可见、可复用特征：年龄感/物种、脸型、五官、发型、体型、服装层级、主色、关键配饰
- 三视图与脸部特写必须同源同角不得变脸：明确写"左右同一角色，脸部特写与三视图面部完全一致，不要改变脸型/五官/发型"
- 禁止写成 9:16 电影静帧、王家卫质感、港风霓虹、分镜动作场面
- 禁止只写"单人肖像/站在场景中"替代参考拼版

### 角色设定图提示词骨架

```text
角色设定图，角色设计参考拼版，3:2 横向构图。
左侧约40%是同一角色的脸部特写，锁定五官、脸型、发型和神态，清晰可复用。
右侧约60%是同一角色的全身三视图并排：正面全身、侧面全身、背面全身。
主体身份：{可见身份锚点}。站姿自然，手脚完整，服装与鞋子清楚。
浅灰中性背景，柔和均匀棚光，干净可比较构图，无复杂场景。
{服装材质/颜色说明}。
左右为同一角色，脸部特写与三视图面部完全一致；不要改变脸型、五官比例、发型、体型、服装配色和关键配饰。
```

### 不合格示例

- 只有正面半身或单人电影静帧
- 角色放在剧情场景里（门廊、雨夜街道等）
- 写了 90s Hong Kong / Wong Kar-wai / 霓虹 / 颗粒但不加设定拼版
- 只写了正面，缺少侧面或背面
- 脸写了两边，但没要求"左右同一角色、脸不漂移"

## 输出格式

输出：用途、工作台资产引用、最终提示词、参考图职责表、禁止变化项、APIMart 参数建议、lint 结果。工作台资产引用不得拼接进最终提示词。
APIMart 参数：
- 角色设定图固定 `model: "gpt-image-2-official"`，`resolution: "1k"`，`quality: "high"`，`size: "3:2"`，`n: 1`
- 其他资产设定图固定 `model: "gpt-image-2-official"`，`resolution: "1k"`，`quality: "high"`，`n: 1`
- 分镜首帧可根据预算选用 `gpt-image-2` 或 `gpt-image-2-official`

创作与编辑方法详见 [生图指南](references/prompting-guide.md)。接口字段详见 [APIMart 接口](references/apimart-api.md)。
