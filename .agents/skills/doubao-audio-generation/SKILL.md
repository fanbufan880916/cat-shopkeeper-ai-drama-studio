# 豆包音频生成 Skill

## 当前工作台的主任务

本工作台不再默认生产逐镜头成品台词，也不提供后期配音兜底。正常对白视频采用：

1. 从最新锁定剧本提取说话角色。
2. 将每个说话人与唯一角色资产绑定。
3. 为每个角色生成可试听的4到5秒单人干声音色候选。
4. 用户试听并锁定一条当前音色。
5. 用户明确点击后，将锁定样本提交Seedance私有音色审核。
6. 获得稳定的 `asset://` 后，由视频提示词按说话顺序引用；样本只提供音色，不复用样本台词。

旧场景母带和台词切片继续保留试听与查档，但不计入声音完成条件，不得作为新对白镜头的默认参考。

## 先判定声音证据

声音风格只能来自最新锁定剧本和创作档案，不能从项目名、工作台名或角色名猜测。必须先确认：

- 年龄和性别表现。
- 时代与地域。
- 语言与口音。
- 音高、音色、语速、音量和咬字。
- 角色性格与当前情绪。

只有剧本明确写出90年代香港电影、港片、港式或香港市井喜剧等设定时，才允许写港式普通话。没有证据时禁止自动加入港式普通话、港风配音腔或年代滤镜。信息不足时标记 `needs_review`，停止真实生成。

不得要求复制具体演员、网红、电影原声或未经授权的真实人物声音。只描述可执行的年龄感、口音和表演特征。

## 音色锚点提示词结构

音色锚点使用纯文本，按以下顺序写成一个自然段：

1. 角色姓名、年龄感和性别表现。
2. 语言、口音、音高和音色。
3. 语速、音量、停顿、咬字和情绪变化。
4. 明确“单人干声、4到5秒、无音乐、无环境声、无音效、无混响”。
5. 一句短样本文本，用于展示声线；不能把它当成镜头成品台词。

参考模板：

```text
小曼是十六到十八岁的女高中生，使用标准普通话，不带港式口音。声线年轻清亮，音高中等偏高，音色柔和但不幼态，语速中等，音量自然，咬字清楚；开头略带犹豫，停顿半拍后变得真诚坚定。只生成小曼一个人的4到5秒干声，不要音乐、环境声、音效、混响或其他人声。小曼轻声说：“今天的阳光，好像比昨天暖一点。”
```

音色锚点禁止：

- 多角色轮流或同时说话。
- 背景音乐、店内底噪、脚步、电话、雨声或任何音效。
- 混响、空间感、远近移动或左右声道表演。
- 超过5秒的长台词、旁白段落或完整剧情对白。
- 只写“自然、稳定、清晰地说”而没有具体声线和表演参数。

生成前运行：

```powershell
python .agents/skills/doubao-audio-generation/scripts/prompt_lint.py <提示词文件> --style auto --kind voice-anchor
```

明确的90年代港片项目才可使用 `--style hk90`。检查失败不得提交真实任务。

## Seedance参考音色职责

音色注册成功后，视频提示词必须把职责写清：

```text
@Audio1 仅提供小曼的年龄感、音色、口音和说话质感，不复用示例台词。
小曼用@Audio1的音色说：“你今天，不一样。”
```

规则：

- 按台词首次出现顺序编号 `@Audio1`、`@Audio2`、`@Audio3`。
- 单镜头最多3个说话角色。
- 每条参考音色不超过5秒，总时长不超过15秒。
- 多角色台词写清先后顺序并禁止重叠。
- 缺少角色绑定、锁定音色或Seedance `asset://` 时禁止生成视频。
- 视频任务保存音色锚点快照；后续更换音色不得改写已提交任务。

## 历史场景母带模式

只有在维护旧数据或用户明确要求制作可试听母带时，才使用旧的多角色结构：

1. 角色声线清单。
2. 声音环境与音乐。
3. 按时间顺序的对白与声音事件。

这种母带可保留试听、切片和查档，但不得再作为正常Seedance对白镜头的完成条件或默认参考。对应检查命令：

```powershell
python .agents/skills/doubao-audio-generation/scripts/prompt_lint.py <提示词文件> --style auto --kind scene-master
```

## 生成前硬检查

音色锚点必须同时满足：

- 只有一个明确角色。
- 有年龄/性别、语言/口音、音高/音色。
- 有语速、音量、停顿、咬字或情绪变化等具体表演方向。
- 明确4到5秒单人干声。
- 明确无音乐、无环境声、无音效、无混响。
- 只有一句短样本文本。
- 不含未经剧本支持的港式普通话。
- 不含具体演员模仿或未授权原声复制要求。

需要维护旧母带结构时，读取 [references/official-template-patterns.md](references/official-template-patterns.md)。
