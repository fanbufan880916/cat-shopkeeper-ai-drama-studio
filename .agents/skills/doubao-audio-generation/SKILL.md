# 豆包音频生成 Skill

## 核心原则

把音频提示词写成“角色声线清单 + 声音环境/音乐 + 按时间顺序的对白与动作”，不要写成一句“请自然地说”。豆包的纯文本模式需要从上下文推断谁在说话、如何说、何时停顿以及声音如何变化。

声音风格不能从工作台名称或角色名称猜测，必须先从锁定剧本判断时代、地域、语言、口音和表演方式。当前《午夜替补》剧本明确是 90 年代香港市井电影喜剧，因此使用港式普通话；其他项目不得自动套用这一风格。可以借鉴时代质感、配音节奏和市井喜剧表演，但不得复制具体演员的声音、台词或电影录音。

## 风格判定

- 剧本出现“90年代香港电影、港片、港式、香港市井喜剧”等明确设定时，使用港式普通话和对应时代音效。
- 纪录片、新闻、采访和旁白项目使用克制清晰的旁白声线，不套用港片表演。
- 动画或漫剧项目使用角色化表演，但仍需按剧本决定普通话、方言或其他语言。
- 现代写实项目使用符合角色背景的自然语言和口音。
- 剧本没有明确声音风格时，先标记为“待导演确认”，不能擅自加入港式普通话、年代滤镜或夸张喜剧腔。

## 官方纯文本结构

严格按以下顺序输出，不输出 JSON、Markdown 列表、模型名称或接口字段：

1. **角色声线清单**：每个角色一行，包含年龄/性别、口音、音高、音色、说话速度、性格和情绪变化。
2. **声音环境与音乐**：一个自然段，说明环境底噪、音乐主乐器、节奏、情绪和对白出现时的混音关系。
3. **对白与声音事件**：按发生顺序逐行写。每一行先写说话人的表演方向，再写中文引号包裹的完整台词；脚步、电话、碰撞、呼吸和停顿也按顺序写入。

参考结构：

```text
男主是中年男性，港式普通话，声线低沉、略带沙哑，语气成熟稳重，情绪失控时语速突然加快。
女主是青年女性，港式普通话，声线清亮带一点沙哑，咬字利落，愤怒时音量逐步提高。

先是雨夜街道的远处车声和店内风扇声。音乐以低音贝斯和合成器 pad 铺底，对白出现时音乐压低，让停顿、呼吸和反转清楚可听，整体保持 90 年代香港市井电影的紧张喜剧感。
男主压低声音，带着不耐烦和一本正经的荒诞感说道：“你先讲清楚，究竟是谁把鞋放进保险箱的？”
女主先停顿半拍，再用港式普通话认真回应：“这不是保险箱，这是本店的消毒柜。”
随后出现一声短促的电话震动和两步急促脚步声。
男主突然拔高声量，语速加快地喊道：“那你为什么收我保险箱的钱！”
```

## 港式普通话表演规则（仅适用于剧本明确指定时）

- 使用“港式普通话”作为口音锚点，不把所有角色写成播音员普通话。
- 喜剧人物要一本正经地说荒唐话；笑点来自语气、停顿、误解和突然升级，不是每句都大喊。
- 铺垫句用中慢速、低音量、尾音收紧；反转句可突然加速或拔高，但必须保持咬字可懂。
- 争吵时按“克制 → 不耐烦 → 情绪升级 → 突然停住”的曲线写，不要连续使用“激动、愤怒、夸张”等空泛形容词。
- 每个角色都必须有独立的声线锚点。禁止只写“男声/女声/自然/清晰”。
- 不要在一个短镜头中安排多人同时说完整台词。复杂对话先做场景母带，再切成单说话人镜头片段。

## 声音环境与音乐

先写可听见的环境，再写音乐，再写对白的混音优先级：

- 环境：雨声、街道车流、店内风扇、霓虹电流、电话震动等；只保留对剧情有用的声音。
- 音乐：明确主乐器、节奏和情绪，例如低音贝斯、合成器 pad、铜管切分、鼓点或突然静音。
- 对白：对白期间降低音乐，不让音乐掩盖咬字；反转落点可以短暂留白。
- 音效：写清楚发生顺序和可见动作，例如“角色转身后出现两步急促脚步声”。
- 不要要求模型复制受版权保护的歌曲或电影原声；只描述时代质感和音乐功能。

## 参考音频规则

- 角色音色母版：每个主要角色一条本人录制或已授权的 10～20 秒干净单人样本。
- 成品台词片段：口型镜头绑定单说话人、完整台词片段，不要绑定整条混音母带。
- 音乐参考：只用于节拍、速度和情绪；最终发行版在后期重新铺授权音乐。
- 参考音频只写职责，不把 URL、资产 ID 或接口字段塞进纯文本提示词；引用关系在工作台外部保存。
- 参考音频有多人、背景音乐、混响或不清晰时，先清理或重新录制。

## 工作台执行流程

1. 从锁定剧本提取所有对白和说话人。
2. 用本 Skill 生成一条场景级纯文本提示词，先生成对白母带。
3. 母带完成后自动按说话人、台词和镜头切成片段；自动边界只是初稿。
4. 人工试听并调整起止时间，确认片段没有串音、断句或环境声污染。
5. 通过的片段绑定到对应镜头，作为 Seedance 口型参考；拒绝的片段不得进入后续镜头。
6. 最终完整母带、对白 stem、音乐 stem 和音效 stem 在后期统一混音。

## 生成前检查

必须同时满足：

- 至少有一条角色声线清单；
- 当前风格为90年代港片时包含“港式普通话”或等价口音描述；其他风格必须写明对应语言、方言或口音；
- 至少有一段环境/音乐方向；
- 每句对白有明确说话人和中文引号；
- 语气方向具体到速度、音量、停顿或情绪变化；
- 声音事件按时间顺序排列；
- 不存在“稳定、清晰、自然地说”这种单独承担全部指导的空泛句式；
- 不包含具体演员模仿、未授权电影音频或受保护歌曲复制要求。

需要详细的官方模板拆解时，读取 [references/official-template-patterns.md](references/official-template-patterns.md)。需要运行自动检查时，执行 `scripts/prompt_lint.py`。
